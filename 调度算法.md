### 最早截至时间优先算法（EDF）

最早截止期限优先（EDF）调度根据截止期限动态分配优先级。截止期限越早，优先级越高；截止期限越晚，优先级越低。根据 EDF 策略，当一个进程可运行时，它应向系统公布截止期限要求。优先级可能需要进行调整，以便反映新可运行进程的截止期限。

```c++
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>

struct Job {
    int id;
    int arrivalTime;
    int burstTime;
    int deadline;

    bool operator<(const Job& other) const {
        return deadline > other.deadline; // 截止期限早的优先级高
    }

    // 构造函数，用于初始化Job
    Job(int id, int arr, int brt, int dl)
        : id(id), arrivalTime(arr), burstTime(brt), deadline(dl) {}
};

void EDF(std::vector<Job>& jobs) {
    std::priority_queue<Job> readyQueue;
    int currentTime = 0;

    for (auto& job : jobs) {
        // 等待直到任务到达
        while (currentTime < job.arrivalTime) {
            // 检查就绪队列中的任务是否错过了截止期限
            while (!readyQueue.empty() && readyQueue.top().deadline <= currentTime) {
                std::cerr << "Error: Missed deadline for job " << readyQueue.top().id << std::endl;
                readyQueue.pop();
            }
            currentTime++; // 如果没有错过截止期限，则递增时间
        }

        // 将任务加入就绪队列
        readyQueue.push(job);
        
        // 如果有新的任务到达，则观察就绪队列是否需要重新排序
        while (!readyQueue.empty()) {
            Job currentJob = readyQueue.top();
            readyQueue.pop();

            if (currentJob.deadline <= currentTime) {
                std::cerr << "Error: Missed deadline for job " << currentJob.id << std::endl;
                continue;
            }

            std::cout << "Executing job " << currentJob.id << " from time " << currentTime
                      << " to time " << currentTime + currentJob.burstTime - 1 << std::endl;

            currentTime += currentJob.burstTime;

            while (!readyQueue.empty() && readyQueue.top().deadline <= currentTime) {
                std::cerr << "Error: Missed deadline for job " << readyQueue.top().id << std::endl;
                readyQueue.pop();
            }
        }
    }
}

int main() {
    // 初始化作业列表
    std::vector<Job> jobs = {
        Job(1, 0, 8, 10),
        Job(2, 1, 4, 6),
        Job(3, 2, 9, 10),
        Job(4, 3, 5, 7)
    };

    EDF(jobs);

    return 0;
}

```



### 最低松弛度优先算法（LLF）

最低松弛度优先算法（Lowest Slack Time First，简称LLF）是一种动态优先级调度算法，它根据进程的剩余处理时间和截止时间来确定优先级，以确保系统能够尽可能快地完成最紧急的任务。在LLF算法中，每个进程都有一个松弛度（Slack Time），表示当前时间到达截止时间的剩余时间。

```c++
#include <iostream>
#include <vector>
#include <algorithm>

// 定义Job结构体
struct Job {
    int id;          // 任务ID
    int arrivalTime; // 到达时间
    int burstTime;   // 服务时间
    int deadline;    // 截止时间
    int remaining;   // 剩余服务时间（初始化为burstTime）

    // 构造函数，用于初始化Job
    Job(int id, int arr, int brt, int dl)
        : id(id), arrivalTime(arr), burstTime(brt), deadline(dl), remaining(brt) {}
};

// SJF作业调度算法函数
void SJF(std::vector<Job>& jobs) {
    int currentTime = 0;
    while (!jobs.empty()) {
        // 找到当前时间已到达且尚未完成的作业
        std::vector<Job*> readyJobs;
        for (auto& job : jobs) {
            if (job.arrivalTime <= currentTime && job.remaining > 0) {
                readyJobs.push_back(&job);
            }
        }

        // 如果没有就绪的作业，则时间前进一个单位
        if (readyJobs.empty()) {
            currentTime++;
            continue;
        }

        // 按照服务时间排序就绪作业
        std::sort(readyJobs.begin(), readyJobs.end(), [](const Job* a, const Job* b) {
            return a->burstTime < b->burstTime;
        });

        // 选择服务时间最短的作业执行
        Job* currentJob = readyJobs.front();
        std::cout << "Executing Job " << currentJob->id << " at time " << currentTime << std::endl;

        // 执行一个单位时间的工作（或减少剩余服务时间）
        currentJob->remaining--;

        // 如果作业完成，则从就绪队列和作业列表中移除
        if (currentJob->remaining == 0) {
            std::cout << "Job " << currentJob->id << " completed." << std::endl;

            // 从就绪队列中移除
            readyJobs.erase(readyJobs.begin());

            // 从作业列表中移除（使用迭代器找到并移除）
            auto it = std::remove_if(jobs.begin(), jobs.end(),
                [&currentJob](const Job& j) { return &j == currentJob; });
            jobs.erase(it, jobs.end());
        }

        // 时间前进一个单位
        currentTime++;
    }
}

int main() {
    // 初始化作业列表
    std::vector<Job> jobs = {
        Job(1, 0, 8, 10),
        Job(2, 1, 4, 6),
        Job(3, 2, 9, 10),
        Job(4, 3, 5, 7)
    };

    // 执行SJF作业调度算法
    SJF(jobs);

    return 0;
}

```



- `Job` 结构体的定义：
  - 定义了包含任务ID、到达时间、服务时间、截止时间和松弛时间的 `Job` 结构体。
  - 构造函数用于初始化任务对象，重载小于运算符以便排序。
- EDF算法的实现过程：
  - 基于最早截止时间优先排序作业。
  - 迭代检查可用作业，并选择最早截止时间的作业执行。
  - 检查作业是否超过截止时间，更新时间继续调度。
- LLF算法的实现过程：
  - 基于最低松弛度优先排序作业。
  - 迭代检查可用作业，并选择最低松弛度的作业执行。
  - 检查作业是否超过截止时间，更新时间继续调度。
- 主函数 `main` 的作用：
  - 初始化不同作业列表传入 EDF 和 LLF 算法以进行作业调度。
  - 依次调用 EDF 和 LLF 算法函数，并输出调度情况。
- 程序运行和输出内容：
  - 运行程序获得的输出包括根据不同算法进行作业调度的结果和可能的错误信息。
  - 输出内容包括作业ID、运行时间点和执行情况。
